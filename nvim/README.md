## Install Instructions

 > Install requires Neovim 0.9+. Always review the code before installing a configuration.

Clone the repository and install the plugins:

```sh
git clone git@github.com:tlj/dotfiles ~/.config/tlj/dotfiles
NVIM_APPNAME=tlj/dotfiles/nvim/.config/nvim nvim --headless +"Lazy! sync" +qa
```

Open Neovim with this config:

```sh
NVIM_APPNAME=tlj/dotfiles/nvim/.config/nvim nvim
```
